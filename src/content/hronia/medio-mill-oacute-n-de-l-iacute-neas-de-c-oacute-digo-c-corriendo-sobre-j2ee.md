Esta interesante entrevista en la imprescindible  The Server Side nos cuenta las aventuras de Infogate, una empresa que desarrolla un plugin para Visual Studio capaz de desplegar aplicaciones en ASP.NET y ADO.NET en servidores de aplicaciones Java, usando para ello Visual Mainwin, una herramienta desarrollada por Mainsoft.  



Curiosamente, el artículo menciona que el servidor de aplicaciones (Tomcat en este caso) corre bajo Linux, lo que me hace sospechar que Mono tiene algo que ver.  Buscando en el documento, lo único que mencionan en la entrevista es que "evaluaron Mono y decidieron no apostar por el código abierto y basarse en el producto de Mainsoft".   Sin embargo, según Novell, utiliza Mono en Visual Mainwin
