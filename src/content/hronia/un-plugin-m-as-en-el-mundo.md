Hoy he terminado mi primera aplicación escrita en Visual C++.  Se trata, para más inri, de un plugin para otra aplicación (claro, de tratarse de una aplicación corriente y moliente no hubiera tenido narices de acabarla).  La aplicación anfitriona es Windows Powerpro. 



¿Que qué es [windowspowerpro.com](http://www.windowspowerpro.com)? Es una aplicación gratuita que permite hacer todo tipo de virguerías con Windows. Desde pintar menús, barras de herramientas, activar temporizadores, ejecutar scripts en un lenguaje propio, usar escritorios virtuales, definir atajos propios de teclado y gestos de ratón.  Incluso notas tipo post-it.  Como dice su autor, PowerPro es capaz de sustituir a 3 o 4 aplicaciones de esas que a los power-users nos gusta tener en Windows desfigurando la interfaz de usuario de forma que sólo nosotros estamos a gusto con nuestro ordenador -un vicio que adquirí con GNU/Linux-.  Y además gratis (aunque no libre)  Hay, por otro lado, una reducida pero voluntariosa comunidad de usuarios en http://www.yahoogroups.com/group/power-pro



Pues bien, héteme aquí que he perpetrado un plugin para que desde el lenguaje de scripting de PowerPro se pueda acceder a las lecturas de los sensores de Motherboard Monitor.   Así puedo hacer un montón de cosas curiosas: desde mostrar en el escritorio de Windows una barra de herramientras transparente con las temperaturas de mi CPU, placa base y la velocidad del ventilador, hasta ejecutar scripts que periódicamente muestreen estos valores y, si sobrepasan ciertos umbrales, activen opciones de emergencia.



He enviado a la lista de Yahoo Groups el aviso y he subido el archivo comprimido, con un bonito readme y el código fuente incluido.  Espero que al resto de forofos les guste el invento -a mí al menos me ha parecido original-.
