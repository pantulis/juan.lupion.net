K-Meleon, el navegador web ligero para Windows basado en el corazón de Mozilla, Gecko, ya tiene  disponible una nueva beta.  El instalador completo para Windows pesa unos 5 Megas.  K-Meleon viene a ser el equivalente a [gnome.org](http://www.gnome.org/projects/epiphany/)para Windows.



Tras instalarlo, se hacen notorias algunas carencias.  No hay barra de búsquedas para Google, ni hay "sidebar" ni nada que se le parezca.  Tiene soporte para navegar usando múltiples pestañas -como hacen [mozilla.org](http://www.mozilla.org/products/firebird/), [opera.com](http://www.opera.com)y las [myie2.com](http://www.myie2.com)para Internet Explorer- , aunque resulta un poco primitivo.  En lugar de usar un auténtico control de solapas, lo que hace es pintar y ocultar ventanas completas según la solapa escogida.   Funcional, pero sin cohetes.



Entonces, ¿qué ventajas tiene K-Meleon?  



Soporta estándares,  porque la versión que salió ayer está basada en el mismo motor que Mozilla 1.5. (Funciona, por ejemplo, la búsqueda de tipo _Type-as-you-go_)

Es ligero.  Vuela.  ¡Maldita sea, corre que se las pela!  Trae incluso un cargador que, si se ejecuta en el arranque de Windows, cargará las DLL más pesadas una única vez y el arranque será aún más veloz.  No es en absoluto necesario hacerlo; incluso sin cargador K-Meleon arranca tan deprisa como Internet Explorer

Es bonito.  Me recuerda a mi versión favorita de Internet Explorer, la 3.0 (en general de infausto recuerdo).  El tema y los iconos escogidos son los mismos y tiene un cierto sabor retro, aunque esto es una preferencia personal.

Es una herramienta para &uuml;ber-nerds.  Tiene su propio sistema de macros y configuración avanzados -aunque, claro, no tan versátil como [xulplanet.com](http://www.xulplanet.com/)-. 

Tiene algunos puntos extra, como bloqueo de popups, gestión de perfiles, API para plugins, soporte de Java y Javascript, 



Se puede descargar de [kmeleon.sourceforge.net](http://kmeleon.sourceforge.net)
